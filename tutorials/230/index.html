2.1 프로젝트 “branch” 변경하기

Tizen 프로젝트의 수정 및 설치해서 사용하기 위해서는 Raspberry PI에 설치된 Tizen Platform 버전에 맞는 branch 를 사용하여야 합니다.

따라서, “git branch” 명령어를 사용하여 branch 들을 확인하고, “git checkout” 명령어를 사용하여 branch 를 변경하여야 합니다.
