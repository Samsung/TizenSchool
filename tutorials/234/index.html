Overview

Overview

This tutorial explains how to use Tizen Peripheral SPI C# API. SPI stands for Serial Peripheral Interface and is a serial bi-directional communication bus, widely used for transferring data between electronic devices.


The application communicates with Accelerometer device (ADXL345) via SPI bus to read measured acceleration along 3-axis.
