Case float

   100.0f.ToPixel()
   : Convert Dp 100 to pixel value. Use it to assign value in NUI pixel properties.

   100.0f.ToDp()
   : Convert pixel 100 to dp value. Use it to get dp value from pixel NUI properties.

 

Support int in the same way.

 

Case Size,
   Size(100.0f, 100.0f).ToPixel()
   : Convert Dp (100x100) size to pixel size.
   Size(100.0f, 100.0f).ToDp()
   : Convert Pixel (100x100) size to dp size.


Support Size2D, Position, Position2D, Rectangle, Extents in the same way.


font have different sizing system which is Point and Pixel.
Pixel is non-density-sensitive value, but Point is density-sensitive.
different between dp and point is baseline dpi, dp use 160dpi for baseline dpi but point use 72 for baseline dpi.

we recommand to use Point size for density-sensitive application development.
I also give extensions for point to pixel or point to dp for user convenience,

text.PointSize = 50.0f.DpToPoint();
text.PointSize = 50.0f.PixelToPoint();
text.PointSize.PointToDp();
text.PointSize.PointToPixel();


ScalingFactor
we plan to add ScalingFactor in model config on the system Infromation,
but we are not ready to do this so temporariliy we provide environment variable.
export NUI_SCALING_FACTOR=value

for the test, I created one file in /etc/profile.d/
cat export NUI_SCALING_FACTOR=2.5 > nui.sh
and rebooted.
